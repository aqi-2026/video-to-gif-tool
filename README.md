# 视频转GIF工具

一个功能强大的批量视频转GIF工具,支持将多种格式的视频文件一键转换为GIF动图。

## 功能特性

- **批量转换**: 一键转换文件夹内所有视频文件
- **多格式支持**: 支持 MP4, AVI, MOV, MKV, FLV, WMV, WebM, M4V 等常见视频格式
- **质量控制**: 提供高、中、低三档质量选项,平衡文件大小和画质
- **友好界面**: 图形化操作界面,操作简单直观
- **实时日志**: 显示转换进度和详细日志信息

## 质量档位说明

| 质量等级 | 帧率 | 缩放比例 | 颜色数 | 文件大小 | 适用场景 |
|---------|------|---------|--------|---------|---------|
| 高质量  | 15fps | 100% | 256色 | 较大 | 需要保持原始画质 |
| 中等质量 | 10fps | 75% | 128色 | 适中(推荐) | 日常使用 |
| 低质量  | 8fps | 50% | 64色 | 较小 | 需要节省空间 |

## 使用方法

### 使用打包后的exe文件

1. 运行 `fin/VideoToGIF.exe`
2. 设置输入文件夹(默认: D:/GIF/start)
3. 设置输出文件夹(默认: D:/GIF/finish)
4. 选择转换质量
5. 点击"扫描视频文件"
6. 点击"开始转换"

### 从源码运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python gui.py
```

## 打包为exe文件

```bash
# 方法1: 使用打包脚本
python build_exe.py

# 方法2: 手动打包
pyinstaller --name=VideoToGIF --onefile --windowed --clean gui.py
```

打包完成后,exe文件将自动复制到 `fin` 文件夹。

## 系统要求

- Windows 7 或更高版本
- Python 3.7+ (仅从源码运行时需要)
- FFmpeg (moviepy会自动处理)

## 注意事项

1. 首次运行时,程序会自动创建输入和输出文件夹
2. 转换大文件时可能需要较长时间,请耐心等待
3. 建议先用低质量模式测试,确认效果后再使用高质量
4. GIF文件大小取决于视频时长、分辨率和质量设置

## 常见问题

**Q: 转换速度很慢怎么办?**
A: 可以尝试降低质量等级,或者将长视频裁剪后再转换。

**Q: 支持哪些视频格式?**
A: 支持常见格式包括 MP4, AVI, MOV, MKV, FLV, WMV, WebM, M4V。

**Q: GIF文件太大怎么办?**
A: 选择"低质量"选项,或者使用视频编辑软件先缩短视频时长。

**Q: 可以自定义输入输出路径吗?**
A: 可以,点击"浏览..."按钮选择任意文件夹。
