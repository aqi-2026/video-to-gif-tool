# 视频转GIF工具 - 使用说明

## 快速开始

### 方式一: 直接使用exe文件(推荐)

1. 打开 `fin` 文件夹
2. 双击运行 `VideoToGIF.exe`
3. 按照界面提示操作即可

### 方式二: 从源码运行

1. 确保已安装 Python 3.7+
2. 安装依赖:
   ```bash
   pip install -r requirements.txt
   ```
3. 运行程序:
   ```bash
   python gui.py
   ```

## 详细操作步骤

### 1. 准备视频文件

将需要转换的视频文件放入输入文件夹(默认: `D:\GIF\start`)

支持的视频格式:
- MP4
- AVI
- MOV
- MKV
- FLV
- WMV
- WebM
- M4V

### 2. 启动程序

双击运行 `VideoToGIF.exe`

### 3. 配置路径

- **输入文件夹**: 存放待转换视频的文件夹
- **输出文件夹**: 转换后的GIF文件保存位置

默认路径:
- 输入: `D:\GIF\start`
- 输出: `D:\GIF\finish`

可以点击"浏览..."按钮自定义路径。

### 4. 选择转换质量

根据需求选择质量档位:

**高质量(文件较大)**
- 帧率: 15 FPS
- 画质: 原始大小
- 颜色: 256色
- 适合: 需要保持最佳画质的场景

**中等质量(推荐)**
- 帧率: 10 FPS
- 画质: 原始的75%
- 颜色: 128色
- 适合: 日常使用,平衡文件大小和质量

**低质量(文件较小)**
- 帧率: 8 FPS
- 画质: 原始的50%
- 颜色: 64色
- 适合: 需要节省磁盘空间或网络传输

### 5. 扫描视频文件

点击"扫描视频文件"按钮,程序会:
- 检测输入文件夹中的所有视频文件
- 显示找到的视频数量和详细信息
- 在日志区域列出每个视频的文件名和大小

### 6. 开始转换

点击"开始转换"按钮,程序会:
- 按顺序处理每个视频文件
- 在日志区域实时显示转换进度
- 显示转换成功/失败的统计信息

转换过程中可以在日志区域看到:
- 当前处理的文件
- 转换进度
- 生成的GIF文件名
- 错误信息(如果有)

### 7. 查看结果

转换完成后:
- 在输出文件夹中查看生成的GIF文件
- 日志区域显示成功和失败的数量
- 弹出提示框显示转换结果

## 文件命名规则

GIF文件会使用原视频的文件名,扩展名改为 `.gif`

示例:
- `my_video.mp4` → `my_video.gif`
- `demo.avi` → `demo.gif`

## 注意事项

### 磁盘空间

- 确保输出文件夹有足够的磁盘空间
- GIF文件可能比原视频更大或更小,取决于视频时长和质量设置

### 转换时间

转换时间取决于:
- 视频文件大小
- 视频时长
- 选择的质量档位
- 计算机性能

通常:
- 短视频(< 10秒): 几秒到几十秒
- 中等视频(10-60秒): 几十秒到几分钟
- 长视频(> 60秒): 可能需要更长时间

### 质量建议

| 场景 | 推荐质量 | 说明 |
|-----|---------|------|
| 表情包制作 | 低质量 | 文件小,加载快 |
| 社交媒体分享 | 中等质量 | 平衡质量和大小 |
| 高质量展示 | 高质量 | 保持最佳画质 |
| 长视频(>30秒) | 低质量 | 避免文件过大 |
| 短视频(<10秒) | 中/高质量 | 文件大小可控 |

## 常见问题解决

### 问题1: 找不到视频文件

**原因**:
- 路径设置错误
- 视频格式不支持
- 文件夹为空

**解决**:
1. 检查输入文件夹路径是否正确
2. 确认视频格式在支持列表中
3. 确保文件夹中有视频文件

### 问题2: 转换失败

**原因**:
- 视频文件损坏
- 编码格式不支持
- 磁盘空间不足

**解决**:
1. 尝试用播放器打开视频,确认视频完整
2. 用视频编辑软件重新编码视频
3. 清理磁盘空间

### 问题3: GIF文件太大

**解决方法**:
1. 选择"低质量"选项
2. 使用视频编辑软件裁剪视频时长
3. 降低原视频分辨率

### 问题4: GIF播放卡顿

**原因**: 文件太大或帧率太高

**解决**:
- 选择更低的质量档位
- 减少视频时长
- 降低原视频分辨率

### 问题5: 程序无响应

**原因**: 正在处理大文件

**解决**:
- 耐心等待,查看日志区域的进度信息
- 如果长时间无响应,关闭程序重试
- 尝试转换更小的视频文件

## 性能优化建议

### 批量转换大量文件

如果需要转换大量视频:
1. 先用1-2个文件测试质量设置
2. 分批次转换,每批20-50个文件
3. 建议在电脑空闲时进行转换

### 处理高清视频

对于高分辨率视频(1080p, 4K):
1. 优先选择"低质量"或"中等质量"
2. 考虑先用视频编辑软件降低分辨率
3. 预留更多时间进行转换

### 加快转换速度

1. 关闭其他占用CPU的程序
2. 选择更低的质量档位
3. 转换前先裁剪视频时长

## 打包说明(开发者)

### 自动打包

使用批处理脚本一键打包:

```bash
build.bat
```

### 手动打包

1. 安装依赖:
   ```bash
   pip install -r requirements.txt
   ```

2. 执行打包脚本:
   ```bash
   python build_exe.py
   ```

3. 或直接使用PyInstaller:
   ```bash
   pyinstaller --name=VideoToGIF --onefile --windowed --clean gui.py
   ```

打包完成后,exe文件会自动复制到 `fin` 文件夹。
