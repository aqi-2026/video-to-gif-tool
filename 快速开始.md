# 快速开始指南

## 使用已打包的exe文件

### 步骤1: 准备视频
将要转换的视频文件放入 `D:\GIF\start` 文件夹

### 步骤2: 运行程序
双击运行 `fin\VideoToGIF.exe`

### 步骤3: 开始转换
1. 点击"扫描视频文件"
2. 选择转换质量(推荐选择"中等质量")
3. 点击"开始转换"
4. 等待转换完成

### 步骤4: 查看结果
在 `D:\GIF\finish` 文件夹中查看生成的GIF文件

## 从源码打包exe文件

### Windows用户(推荐)

双击运行 `build.bat`,自动完成所有步骤

### 手动打包

1. **安装Python**
   - 下载并安装 Python 3.7 或更高版本
   - 下载地址: https://www.python.org/downloads/

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **执行打包**
   ```bash
   python build_exe.py
   ```

4. **查看结果**
   打包完成后,exe文件位于 `fin\VideoToGIF.exe`

## 常见问题

**Q: 我的视频在哪个文件夹?**
A: 默认输入文件夹是 `D:\GIF\start`,你可以在程序中修改。

**Q: GIF文件保存在哪里?**
A: 默认输出文件夹是 `D:\GIF\finish`,你可以在程序中修改。

**Q: 选择哪个质量比较好?**
A: 推荐选择"中等质量",平衡文件大小和画质。

**Q: 转换需要多长时间?**
A: 取决于视频大小和时长,通常几秒到几分钟不等。

**Q: 支持哪些视频格式?**
A: 支持 MP4, AVI, MOV, MKV, FLV, WMV, WebM, M4V 等。
