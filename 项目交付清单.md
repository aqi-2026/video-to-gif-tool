## 🎯 核心功能

### 1. 批量视频转GIF
- 一键转换文件夹内所有视频
- 支持MP4, AVI, MOV, MKV, FLV, WMV, WebM, M4V

### 2. 三档质量控制
- **高质量**: 15fps, 100%分辨率, 256色
- **中等质量**: 10fps, 75%分辨率, 128色 (推荐)
- **低质量**: 8fps, 50%分辨率, 64色

### 3. 图形用户界面
- 友好的PyQt5界面
- 实时日志显示
- 进度条显示
- 路径自定义

### 4. 自动化打包
- 一键打包成exe
- 自动处理依赖
- 输出到fin文件夹

## 🚀 快速开始

### 用户使用(需在Windows上先打包)

1. 放入视频到 `D:\GIF\start`
2. 运行 `fin\VideoToGIF.exe`
3. 点击"扫描视频文件"
4. 选择质量,点击"开始转换"
5. 在 `D:\GIF\finish` 查看GIF

### 开发者打包(在Windows上执行)

**方式1: 自动打包**
```cmd
build.bat
```

**方式2: 手动打包**
```cmd
pip install -r requirements.txt
python build_exe.py
```
