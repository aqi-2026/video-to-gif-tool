## 技术实现

### 核心技术栈
- **Python 3.7+**: 主要开发语言
- **MoviePy**: 视频处理和GIF生成
- **Pillow**: 图像处理支持
- **PyQt5**: 跨平台GUI框架
- **PyInstaller**: 打包工具

### 质量控制参数

| 质量 | 帧率 | 缩放 | 颜色数 | 特点 |
|-----|------|------|--------|------|
| 高  | 15fps | 100% | 256色 | 最佳画质,文件较大 |
| 中  | 10fps | 75%  | 128色 | 推荐使用,平衡质量 |
| 低  | 8fps  | 50%  | 64色  | 文件较小,节省空间 |

### 特色功能

1. **智能路径管理**
   - 自动创建输入输出目录
   - 支持自定义路径
   - 默认路径: D:\GIF\start 和 D:\GIF\finish

2. **用户友好**
   - 直观的图形界面
   - 实时进度显示
   - 详细的日志信息
   - 转换完成提示

3. **高效处理**
   - 异步转换,不阻塞界面
   - 支持批量处理多个文件
   - 自动跳过损坏的文件

## 使用流程

### 最终用户使用流程
```
1. 准备视频文件 → 放入D:\GIF\start
2. 运行程序 → 双击fin\VideoToGIF.exe
3. 扫描文件 → 点击"扫描视频文件"
4. 选择质量 → 选择高/中/低质量
5. 开始转换 → 点击"开始转换"
6. 等待完成 → 查看实时日志
7. 查看结果 → 在D:\GIF\finish中获取GIF
```

### 开发者打包流程
```
1. 准备环境 → Windows + Python 3.7+
2. 安装依赖 → pip install -r requirements.txt
3. 执行打包 → 双击build.bat
4. 获取exe → 在fin文件夹中获取VideoToGIF.exe
```
